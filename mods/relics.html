About

Relics is a Mindustry mod adding lost content from Mindustry Classic that was removed in updates. This is mainly turrets, as most other removed content has equivalents in the current game.

Features

  • Turrets from Mindustry Classic, remade to fit in the current game
  • A modified tech tree to give access to the new turrets

Install

Via GitHub

  1. Go to the latest release of the mod
  2. Download Relics-Mod.jar from the downloads section
  3. Open Mindustry, click "Mods", then click "Import File"
  4. Find the file you downloaded earlier and select it
  5. Restart Mindustry to finish loading the mod

Via Mod Browser

  1. Open Mindustry
  2. Click "Mods" and then open the Mod Browser
  3. Search for "Relics"
  4. Click the result that has the author "[Error_in_Code]"
  5. Click "Install" and then restart Mindustry
